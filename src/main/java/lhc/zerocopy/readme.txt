我们将磁盘上的文件读取到内存里发送给用户，这种场景在开发中很多，fileInputStream用流的形式

但凡io操作，调用操作系统API，
传统数据操作：
用户空间发送读请求到内核空间操作系统，切换到内核模式，从内核空间发送读请求到磁盘（DMA）
读取到内存空间缓冲区（第一次拷贝），将内核空间缓冲区复制到用户空间缓冲区（第二次拷贝）
执行业务逻辑代码
从用户空间拷贝到内核缓冲区（第三次），从内核缓冲区写到磁盘或者socket（第四次），从socket拷贝到协议栈（第五次）


mmap：
从磁盘拷贝到内核，内核拷贝到socket，从socket拷贝到协议栈



sendFile:零拷贝

用户空间向内核空间发送sendfile请求，内核空间向磁盘发送data向磁盘读

拷贝到内核缓冲区，再拷贝到协议栈
